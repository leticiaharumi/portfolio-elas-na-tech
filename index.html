Quem sou

Oi! :)

Me chamo Leticia, mas podem me chamar de Lê ou Haru. Tenho 31 anos e moro em São Paulo/SP desde os meus 8 anos. Sou formada em Arquitetura e Urbanismo (2017) e resolvi, durante a pandemia, me aventurar em uma área nova! E agora, também estou formada em Análise e Desenvolvimento de Sistemas (dezembro de 2022).

Elas na Tech

Turma 07 │ 2023

A minha jornada no Elas Na Tech, teve início a partir da indicação do curso através de algum grupo de whatsapp. Acabei acessando o site, e ficando extremamente interessada! Como conhecimento nunca é demais, me inscrevi e fiquei torcendo para conseguir uma vaga e ter a oportunidade de adquirir mais conhecimento!

Desafio

Aprender novos conteúdos é sempre muito desafiador, mesmo tendo algum conhecimento sobre HTML e CSS, sempre tive uma maior dificuldade quanto ao Javascript. Mas as aulas e a dedicação da professora Gil e da monitora Vanessa, sempre me motivaram a me esforçar para aprender cada dia mais, mesmo que ainda continuasse tendo uma maior dificuldade.

Conquista

Durante o curso, pude ampliar meus horizontes e aprimorar minhas habilidades na área de desenvolvimento web. Esta experiência, foi crucial para meu crescimento como profissional da tecnologia. Adquiri uma sólida compreensão em lógica de programação, permitindo-me abordar desafios complexos de forma estruturada e eficiente. Aprendi a criar páginas web intuitivas e visualmente atrativas com HTML e CSS. Além disso, familiarizei-me com Git e GitHub, o que me possibilitou compartilhar de forma facilitada os projetos que desenvolvi. Ao concluir o curso, sinto que estou preparada para enfrentar os desafios e aproveitar as oportunidades no mundo do desenvolvimento web. Agradeço à equipe do Elas na Tech, por proporcionar uma educação excepcional, e abrir portas para um novo futuro na área de tecnologia.

Projetos

Desafio 06

O site desenvolvido para o desafio da aula 6 apresenta uma estrutura fundamental composta por um arquivo index.html e CSS. Ele inclui um cabeçalho, demonstração da hierarquia dos títulos, parágrafos, listas ordenadas e não ordenadas, além de um link. Utilizando CSS para a estilização, foram feitas modificações simples, como ajustes de cores, tamanhos de fonte, margens, e a adição de um efeito hover ao link.

Desafio 07

O site criado para a resolução do desafio 7 foi elaborado com a aplicação de tags semânticas essenciais, tais como: header, nav, main, section, article e footer. Nele, foi incorporado botões com um efeito hover, e seis seções distintas dentro da tag principal. Cada uma dessas seções inclui um título, um gif com efeito hover e um texto correspondente.

Desafio 08

O site elaborado para o desafio da aula 8 apresenta uma estrutura HTML básica, composta por um header fixo no topo com um logotipo e três ícones na barra de navegação. No elemento principal (main), encontra-se uma "section" chamada "container", que contém nove divisões (divs). Em cada uma dessas divisões inclui uma imagem com bordas arredondadas e, um título que aparece com o efeito hover. No final da página, há um rodapé (footer) simples. O CSS faz uso do FLEXBOX para organizar os elementos da página HTML.

Desafio 09

Neste desafio, desenvolvi uma página de receitas utilizando uma estrutura HTML básica. Esta inclui um cabeçalho, uma barra de navegação (nav), uma seção denominada "cards_container" contendo dez divisões (divs), e um rodapé (footer). Dentro de cada divisão do "cards_container", estão uma imagem e uma div contendo informações sobre a receita (como o nome do prato e um link para a receita em si). Além disso, são exibidos dois ícones para representar as opções de "curtir" e compartilhar. No aspecto do estilo (CSS), utilizei media query para garantir a responsividade da página.

Desafio 10

Neste desafio com o tema Studio Ghibli, foi criado uma página com uma estrutura HTML básica. Nela inclui um cabeçalho (header), barra de navegação (nav), corpo principal (main) com uma seção de imagem de fundo, texto informativo e botão para mostrar ou ocultar o trailer. Na mesma seção principal, há outra área com quatro artigos em formato de card, apresentando informações sobre filmes do Studio Ghibli. Cada card tem um ícone com efeito de zoom, um breve texto e um botão. Além disso, há um rodapé. Utilizamos o arquivo "normalize.css" para padronizar os elementos HTML.

Desafio 11

No desafio 11, foi criado um programa de verificação de número. Onde é solicitado ao usuário um número inteiro e este, verifica se o número atende aos seguintes critérios, sem nenhuma exceção: é um número par; é um número positivo (maior que zero); é um número menor que 100. Se o número atender a todos os critérios descritos anteriormente, o programa deverá imprimir a mensagem "O número atende aos critérios.", caso contrário, imprimirá a mensagem "O número não atende aos critérios". A verificação é feita através de uma função criada no Javascript.

Desafio 12

Neste desafio, desenvolvemos uma calculadora capaz de realizar operações fundamentais (soma, subtração, multiplicação e divisão) entre dois números. Além disso, incluímos um botão de reset e uma área para exibição dos resultados. As operações são processadas por funções Javascript.

Desafio 13

O desafio da aula 13 consiste em um site responsivo com a temática do Studio Ghibli. Sua estrutura HTML inclui elementos essenciais como o cabeçalho (header), barra de navegação (nav), e um corpo principal (main) com um fundo, informações e um rodapé (footer). No desenvolvimento deste projeto, empregamos o arquivo "normalize.css". Além disso, implementamos JavaScript para criar um menu hamburger, que será visível apenas em telas de menor dimensão.

Desafio 14

Este desafio é uma extensão do projeto da aula 13, onde toda a estrutura inicial foi mantida e apenas adicionamos um botão que direciona para a página da galeria. Para criar a página da galeria, utilizamos Bootstrap para criar um carrossel responsivo. Este carrossel apresenta imagens que ocupam toda a área disponível, junto com setas laterais, indicadores e uma legenda simples. Além disso, incluímos um botão na parte superior esquerda para retornar à página principal.
